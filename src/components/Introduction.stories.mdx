# Welcome to React Floating Icons Storybook

Storybook is usefull to build UI components in isolation from your app's business logic, data, and context.
That makes it easy to develop hard-to-reach states. Save these UI states as **stories** to revisit during development, testing, or QA.

Browse example stories now by navigating to them in the sidebar.
View their code in the `src/components` directory to learn how they work.
We recommend building UIs with a [**component-driven**](https://componentdriven.org) process starting with atomic components and ending with pages.

        <p>`position: object` - Position of the item.</p>
        <p>
          `size?: object` - Custom size of the item. Usefull to resize big
          images.
        </p>
        <p>
          `animationDuration?: string` - Custom animation duration. 3 seconds by
          default.
        </p>
        <p>
          `className?: string` - ClassName for the image or any custom html
          passed.
        </p>
        <p>`src?: string` - Source/url of the image passed.</p>
        <p>
          `isNotImage?: boolean` - If you want to pass a custom HTML set this
          prop to true. Defaults to false.
        </p>
        <p>
          `altText?: string` - Description of the image (**alt** attribute for
          **img** HTML tag).
        </p>
        <p>
          `content?: string or ReactNode` - Custom string or HTML instead of
          image (**src**).
        </p>
        <p>
          `onClick?: () => unknown` - click handler. Not recommended given the
          items are not static.
        </p>
      </td>
    </tr>
    <tr>
      <td>backgroundContent (optional)</td>
      <td>object</td>
      <td>
        The background of the container with the following keys:
        <p>
          `imagePath?: string` - If you want an image as background, send the
          url.
        </p>
        <p>
          `color?: string` - If you want some color send it as you would apply
          to the css background-color.
        </p>
      </td>
    </tr>
    <tr>
      <td>sizeWrapper (optional)</td>
      <td>object</td>
      <td>
        The size of the container with the following keys:
        <p>`width: string` - width of the container. Defaults to 100%.</p>
        <p>`height: string` - height of the container. Defaults to 400px.</p>
      </td>
    </tr>
